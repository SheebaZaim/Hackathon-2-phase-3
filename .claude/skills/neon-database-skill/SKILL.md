# Neon Database Agent Skill

## Instructions
1. Provision PostgreSQL databases on Neon.  
2. Configure secure connections, credentials, and environment variables.  
3. Create, optimize, and index tables and queries.  
4. Implement backup, recovery, and failover strategies.  
5. Monitor database performance and optimize queries for production.  
6. Manage migrations and schema changes safely.  
7. Ensure data integrity, consistency, and security.  

## Examples
- Set up a Neon DB instance for a FastAPI project.  
- Optimize query performance using indexes and EXPLAIN plans.  
- Configure environment variables for secure DB credentials.  
- Perform schema migration with minimal downtime.
