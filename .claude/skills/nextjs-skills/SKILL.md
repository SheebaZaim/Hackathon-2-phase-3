# Next.js Frontend Agent Skill

## Instructions
1. Generate responsive layouts using Next.js and Tailwind CSS.  
2. Implement pages, routing, dynamic data fetching, and client-side interactivity.  
3. Connect frontend to FastAPI backend APIs securely.  
4. Build reusable components, forms, and UI elements.  
5. Optimize frontend for SEO, accessibility, and performance.  
6. Integrate authentication flows (signup, login, RBAC).  
7. Maintain consistent styling and design system across the application.  

## Examples
- Create a dynamic `/products` page fetching data from FastAPI API.  
- Build a reusable `Navbar` component with login/logout buttons.  
- Implement responsive checkout form with input validation.  
- Add client-side authentication logic connected to Auth Agent.
